Fun CSS Properties

5/16/20

While working on the heading for this site, I came across some cool CSS - some of which I never would have imagined we'd have on the web!

Cursor Fun

The basic cursor: pointer; property is probably the best best known modification of the pointer style. But there's are a lot more. Here are a few (hover over the list items):

  • help
  • wait
  • crosshair
  • not-allowed
  • zoom
  • grab

Flexin'

The full range of abilities offered by flexbox are still a mystery to me, but one nifty property that helped me cut down on my media queries was flex-wrap: wrap;. Resize the box below to watch the inner div's adapt and change (resize with right orange border):

Viewport min-width

I'm used to throwing this meta tag: 

<meta name="viewport" content="width=device-width, initial-scale=1" />
in pretty much every head I write these days, as a bare mimimum for mobile web support. But by adding ,width=300 onto the end, you can also set the min width of the viewport, at which point everything in the viewport will scale down proportionally, no more line breaks or layout changes.

Who knew!
